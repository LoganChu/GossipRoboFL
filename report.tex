\begin{abstract}
Decentralized robot swarms must learn collaboratively without a central
coordinator.
We built \sys, a fully peer-to-peer federated learning system for 50-robot
swarms operating on dynamic, proximity-based communication graphs.
Robots train locally on heterogeneous image data and share model weights only
with physical neighbors via gossip.
Using the Clipped Gossip Byzantine-robust aggregator, \sys sustains 75.7\%
classification accuracy on CIFAR-10 under 20\% malicious robots---within
6~pp of centralized FedAvg with no server---and converges even at spectral
gaps as low as $\lambda_2{=}0.055$.
\end{abstract}

\section{Goals}

Traditional federated learning (FedAvg~\cite{mcmahan2017}) relies on a
central parameter server that is both a communication bottleneck and a single
point of failure.
Robot swarms operating in GPS-denied or infrastructure-free environments
cannot assume persistent server connectivity.
\sys pursues four distributed-systems goals:

\begin{enumerate}
\item \textbf{Serverless coordination.}
Replace the central aggregator with pure peer-to-peer gossip averaging: each
robot pushes its model to $k{=}4$ physical neighbors per round.
\item \textbf{Byzantine fault tolerance.}
Withstand sign-flip adversaries corrupting up to 30\% of the swarm using the
Clipped Gossip aggregator~\cite{he2022}.
\item \textbf{Topology sensitivity.}
Characterize how communication range $r$ and algebraic connectivity
$\lambda_2$ (second eigenvalue of the graph Laplacian) govern convergence.
\item \textbf{Data heterogeneity robustness.}
Maintain performance under Dirichlet-partitioned non-IID data
($\alpha \in [0.1, 10.0]$) and straggler clients with variable local epochs.
\end{enumerate}

\section{System Design}

\subsection{Gossip Federated Learning Protocol}

\sys implements synchronous gossip FL over a time-varying communication
graph.
Each round proceeds in three phases: (1)~every robot independently performs
local SGD on its private data shard; (2)~each robot transmits its current
model weights to $k{=}4$ randomly sampled neighbors within communication
range; and (3)~each robot aggregates the received models using the selected
aggregation rule.
The communication graph is reconstructed every 5~rounds as robots execute a
Gaussian random walk ($\sigma{=}0.05$ per step on a unit-square arena),
capturing the dynamic encounter topology of a physical swarm.

\subsection{Aggregation Rules and Byzantine Resilience}

Two aggregation strategies are evaluated.
\emph{Mean averaging} computes a simple weighted average of all received
models, providing no protection against adversarial updates.
\emph{Clipped Gossip}~\cite{he2022} clips each
neighbor's model delta to an $L_2$ ball of radius $\tau$ before averaging,
bounding the influence of any individual peer and limiting honest-model drift
to $O(\tau)$ regardless of the Byzantine fraction $f$.
The clipping radius $\tau$ is auto-calibrated each round as the median
pairwise model distance among received updates.

\subsection{Fault and Topology Models}

Byzantine agents execute \emph{sign-flip} attacks: upon completing local
training, they negate all model parameters before broadcasting, maximally
disrupting honest robots' aggregation.
The communication topology is modeled as a random geometric graph on the
unit square: two robots can exchange models if and only if their Euclidean
distance is within communication range $r$.
The algebraic connectivity $\lambda_2$---the second eigenvalue of the graph
Laplacian---serves as a quantitative proxy for gossip mixing speed; theory
predicts convergence rate $O(1/\lambda_2)$.

\subsection{Learning Setup}

Each robot trains a compact 3-block convolutional network
(${\approx}487$k parameters, ${\approx}2$\,MB per gossip message) on a
private CIFAR-10 partition drawn from a Dirichlet($\alpha$) distribution
over class labels.
Lower $\alpha$ produces more severe label skew: at $\alpha{=}0.1$ each
robot holds examples from ${\approx}1$--2 of 10 classes.
Straggler behavior is modeled by sampling each robot's local epoch count
uniformly from $\{1,\ldots,5\}$ per round.
All experiments use $N{=}50$ robots, fanout $k{=}4$, and 200 gossip rounds
on GPU hardware.
The centralized FedAvg baseline~\cite{mcmahan2017} uses the same model and
data partitioning with a parameter server aggregating all clients
synchronously each round.

\section{Research Questions}

\textbf{Q1:} How do Byzantine fraction $f$ and aggregation algorithm interact
to determine accuracy under a sign-flip attack?

\textbf{Q2:} How does communication range $r$ (and spectral gap $\lambda_2$)
govern gossip convergence under dynamic robot mobility?

\textbf{Q3:} How resilient is gossip FL to Dirichlet data heterogeneity
($\alpha$) and client straggler effects?

\section{Experiments \& Results}

Unless otherwise noted, experiments use Dirichlet $\alpha{=}0.5$ and
20\% Byzantine robots.
The centralized FedAvg baseline achieves \textbf{81.5\%} accuracy.

\subsection{Q1: Byzantine Robustness}

Table~\ref{tab:byz} reports final honest-robot accuracy across Byzantine
fractions under a sign-flip attack.
Mean averaging collapses to near-random accuracy (${\approx}10\%$) at
$f{\geq}20\%$---a failure mode that directly mirrors Byzantine consensus
impossibility without sufficient redundancy~\cite{lamport1982}.
Clipped Gossip degrades gracefully, retaining 52.3\% at $f{=}10\%$
under the Byzantine sweep's $\tau$ calibration.
With the default $\tau$ setting used in the topology and heterogeneity
ablations, Clipped Gossip sustains \textbf{75.7\%} at $f{=}20\%$: a
\textbf{64.7~pp} improvement over Mean at the same fault level, within
6~pp of the centralized FedAvg ceiling (81.5\%).

\subsection{Q2: Network Topology and Spectral Gap}

Table~\ref{tab:topo} sweeps communication range $r$ under dynamic mobility.
Spectral gap $\lambda_2$ grows 170${\times}$ (from~$0.055$ to~$9.29$) as
$r$ increases from~$0.2$ to~$0.6$, yet final accuracy shifts by only
\textbf{3.4~pp}.
At $r{=}0.2$ ($\lambda_2{=}0.055$), the random-geometric graph is
near-disconnected yet gossip still converges to 74.1\%---demonstrating
\emph{partition tolerance} analogous to eventual consistency.
Accuracy peaks at $r{=}0.5$ ($\lambda_2{=}5.1$) and shows no further
improvement with densification, consistent with the $O(1/\lambda_2)$
mixing-time bound saturating once $\lambda_2 \gg 1$.

\subsection{Q3: Data Heterogeneity}

Table~\ref{tab:noniid} varies Dirichlet $\alpha$.
The \textbf{15.4~pp} accuracy gap from $\alpha{=}0.1$ to $\alpha{=}10.0$
confirms that data distribution---not topology---is the dominant performance
driver in gossip FL.
At $\alpha{=}0.1$, each robot sees ${\approx}1$--2 of 10 classes; divergent
local gradients overwhelm gossip averaging faster than SSClip can reconcile
them.
Stragglers---robots whose local epoch count is drawn uniformly from
$\{1,\ldots,5\}$ per round rather than fixed at 3---perform within
\textbf{0.3~pp} of homogeneous clients, suggesting $\tau$-bounded clipping
absorbs the resulting gradient staleness without requiring synchronization.

\textbf{Communication cost:} At $k{=}4$, $N{=}50$, each round transfers
${\approx}400$\,MB of model weights---linear in $k$, with no server I/O
bottleneck, compared to FedAvg's star topology where server bandwidth scales
with $N$.

\section{Conclusion and Debrief}

\textbf{Key findings.}
Clipped Gossip enables Byzantine-robust gossip FL within 6~pp of centralized
FedAvg with zero server infrastructure.
Network topology matters only at the sparse extreme ($\lambda_2 {<} 0.5$);
beyond this threshold, convergence is effectively topology-agnostic.
Data heterogeneity ($\alpha$) is the dominant accuracy driver; straggler
effects are negligible.

\textbf{Distributed systems connections.}
Gossip protocols are a classical DS primitive for epidemic broadcast and
push-sum averaging~\cite{kempe2003}.
\sys demonstrates their application in Byzantine fault-tolerant model
averaging---a direct analog of Byzantine consensus in asynchronous
message-passing networks.
The spectral gap $\lambda_2$ serves as the network's algebraic proxy for
consensus round complexity, connecting graph theory directly to convergence
guarantees.

\textbf{Obstacles encountered.}
\begin{enumerate}
\item \textit{Spectral gap on disconnected graphs.}
NetworkX's \texttt{algebraic\_connectivity} uses an internal
\texttt{tracemin\_lu} solver that returns NaN on disconnected graphs.
We replaced it with \texttt{numpy.linalg.eigvalsh} on the explicit Laplacian,
returning $\lambda_2{=}0$ for disconnected components.
\item \textit{Experiment scale.}
200~rounds $\times$ 50~robots $\times$ 14~conditions required
${\approx}18$~GPU-hours total.
\end{enumerate}

\textbf{Future work.}
Three extensions are immediately actionable:
(1)~\emph{push-sum gossip}~\cite{kempe2003} for provably correct weighted
averaging under dropped messages and variable delays;
(2)~\emph{SCAFFOLD control variates}~\cite{karimireddy2020} to correct
client drift from severe non-IID partitions without synchronization;
(3)~physical deployment on differential-drive robots with IEEE~802.11p
ad-hoc communication to validate simulation assumptions.
